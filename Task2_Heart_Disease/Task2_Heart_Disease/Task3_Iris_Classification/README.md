# Iris Flower Classification

This project classifies iris flowers into three species based on petal and sepal measurements.

## Dataset
- Iris Dataset (built-in from scikit-learn)

## Algorithm Used
- Logistic Regression

## Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Tools Used
- Python
- Pandas
- Scikit-learn
